Signup Process for Optics Website:

Customers and Retailers should follow the signup process on the Optics website to access the shop page.

Optics eCommerce Website:

Users can select products from the Optics lens shop to place an order.

Optics Product Categories:

the eCommerce product categories can be created and allowed to be displayed on the website in three layers.

Customized Sale Process:

Additional cart lines will be added if the user selects lens color or lens coating against each product.

Lens Product:

Lens products can be created by selecting the product type option as Lens.

Lens Color:

Lens products can be created by selecting the product type option as Color.

Lens Coating:

Lens products can be created by selecting the product type option as Coating.

Lens Diameter:

Lens Diameter can be added with max and min value of sphere, cylinder, addition and prism on lens product.

Lens Individual Parameters:

Lens products can be created by selecting the product type option as Coating.

Lens Diameter:

Lens Diameter can be added with max and min value of sphere, cylinder, addition and prism on lens product.

Lens Individual Parameters:

Lens Individual Parameters e.g Pupil Distance, Back Vertex Distance, Pantoscopic Angle, Bow Angle, Near Object Distance, Progression Length, Inset, Consumer-Oriented Design, Relax Factor e.t.c. can be added to the lens product. You can allow it to be displayed on eCommerce Website.

Custom Promotion and Coupon Program:

Promotion or Coupon programs can be applied to optics products through domains in each program. If the filtered product appears in the cart line promotion or coupon program will be applied to the product along with color or coating if it exists.
